NAME

ocr-learn - create font data files  

SYNOPSIS

ocr-learn font image[...]  

DESCRIPTION

The ocr-learn program generates font data files for the ocr program. The operands consist of a font name for data storage, followed by the filenames of any number of BMP images as samples of the font. Each image must contain the same characters in the same order, and in the desired font. The characters in the images must be the following string, or any initial part of it.
ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789.,?!/@_
 

EXAMPLES

Generate data for the FreeSans font using samples in a variety of sizes:
ocr-learn freesans freesans36.bmp freesans48.bmp freesans72.bmp
 

FILES

Font information is stored in files in the font directory in the same directory as the executable. These files have the suffix .font Font data files can be used with the ocr program.
