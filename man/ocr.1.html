NAME

ocr - recognize characters in a bitmap image  

SYNOPSIS

ocr font image  

DESCRIPTION

The ocr program identifies text in a bitmap image. Given a BMP image and a supported font name, the text recognized in the image will be printed.  

EXAMPLES

Identify text in the FreeSans font: ocr freesans somefile.bmp  

FILES

Font information is retrieved from files in the font directory in the same directory as the executable. These files have the suffix .font Font data files can be generated using the ocr-learn program. Several font data files are included with the ocr programs.  

BUGS

The algorithm for splitting letters relies on the existence of blank space between letters. Issues will be encountered with fonts whose letters overlap horizontally. Detection of space characters is also somewhat crude. With certain fonts, too few or too many space characters may be detected.
